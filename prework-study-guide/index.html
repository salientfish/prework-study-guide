CSS

  • Cascading Style Sheets.
  • Style sheet language used for describing the presentation of a document written in markup language such as HTML or XML.
  • Enables separation of content and presentation, including layout, colors, and fonts.
  • Improves accessibility.
  • Feasible to present the same markup page in different styles for different rendering methods.
  • A margin indicates how much sapce we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • Period preceding card designates this as a class selector.
  • Class applies to all elements.
